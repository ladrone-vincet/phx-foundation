# Introduction
Hello there!
It is little educational Phoenix app implementing basic OAuth based authorization, simple CRUD via REST API and also Foundation CSS framework.

## Used technologies
    * Phoenix & Elixir
    * Foundation.css
    * PostgreSQL & Ecto 
    * more minor Phoenix modules
